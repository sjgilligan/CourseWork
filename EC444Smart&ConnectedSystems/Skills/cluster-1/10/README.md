Author: Sebastian Gilligan

Date: 2024-09-16

### Summary

Created a program that receives input over UART, and toggles between 3 modes with an 's'. Mode 1: toggle LED with 't', Mode 2: echo strings, Mode 3: echo a decimal number in hexidecimal.
